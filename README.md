Este projeto consiste em um sistema desenvolvido em Java para o gerenciamento de estoque de um armazém logístico de e-commerce. A aplicação opera via console e realiza a persistência dos dados inteiramente em memória, sem a necessidade de um banco de dados externo.

As principais funcionalidades incluem o cadastro de novos produtos, atualização de quantidades em estoque, listagem geral e pesquisa por código, nome ou categoria. Para otimizar o desempenho, o sistema utiliza a estrutura de dados HashMap, que garante acesso imediato nas buscas por código, além de ArrayList para manipulação das listas.

O projeto também conta com um sistema de ordenação interativo. Foi implementado manualmente o algoritmo Bubble Sort para organizar os produtos por preço, permitindo ao usuário escolher entre a visualização crescente ou decrescente através de menus de navegação.
